This folder contains example of C# projects to remotely control the Skydel GNSS simulator.
You can find a list of all existing commands in the Skydel-SDX API directory: see file "Documentation.txt"

* WINDOWS *
All C# examples for remote commands and HIL are in the visual Studio 2013 solution "SdxExamples.sln"

The SdxApi project depends on JSON.net
Please use NuGet Packet Manager to install it.

* OTHER PLATFORMS *
Not supported


* HOW TO RUN EXAMPLES *
1- Compile the example
2- Start Skydel and close the splash screen once the licence has been verified.
3- Start the example executable
